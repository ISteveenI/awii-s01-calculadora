Esto haría que el código sea más difícil de leer, entender y mantener, especialmente para alguien que lo vea por primera vez. También aumentaría la probabilidad de cometer errores al modificarlo.
## 13. Notaste que el código para 'pedir un número al usuario' o 'imprimir el resultado' se repite varias veces. ¿No sería mejor escribirlo una sola vez y reutilizarlo en muchos lugares?
Sí, noté que hay partes del código que se repiten, como pedir números al usuario o imprimir resultados.
Creo que sería mejor crear funciones reutilizables para estas tareas, de manera que el código sea más limpio, organizado y fácil de mantener. Esto también evita duplicar lógica innecesariamente.
